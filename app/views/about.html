A propos du projet

But synthétique du projet :

Gestion d'un annuaire de profiles via un service REST: (l'image est cliquable et envoit sur les Projects).

  • Utilisateurs, Projets, Roles
  • Créations, Modification, Suppression d'éléments
  • Affichage: liste, filtre, tri, recherche

Ce que mon application est capable de faire :

Mon application Angular "Gestion d'un annuaire de profiles" est d'interagir avec le service REST (l'image est cliquable et envoit sur les Users). Elle est capable les actions suivantes pour les Utilisateurs et les Projets :

  • On peut créer un utilisateur ou un projet en remplissant un formulaire avec les informations correspondantes
  • On peut afficher/masquer la liste de tous les utilisateurs et de tous les projets
  • On peut mettre à jour les utilisateurs et les projets
  • On peut supprimer un/des utilisateurs ou un/des projets
  • On peut rechercher un utilisateur ou un projet parmi la liste de tous les utilisateurs et des projets
    • La visualisation d'un utilisateur permet de voir les projets auxquelles celui-ci à participer et son rôle
    • La visualisation d'un projet permet de voir les utilisateurs et leurs rôle dans ce projet
