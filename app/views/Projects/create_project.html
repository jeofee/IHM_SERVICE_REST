Création d'un projet Vous avez plusieurs champs à remplir notammment le titre du projet, la petite description et l'année de création du projet. Vous avez également deux champs à selectionner : les utilisateurs qui ont contribués au projet ainsi que leurs rôles dans ce projet Remplissez les champs libres puis cliquez sur le bouton "création du projet". Remarque : Afin de vérifier que le projet a bien été créé nous vous conseillons de vous diriger vers l'onglet "All Projects" afin de verifier que le projet que vous avez crée est bien dans la base de données.
